# README

Stack
-----

* Ruby 2.3.7
* Rails 5.0.7.2
* postgresql on the backend

System dependencies
-------------------

faker (1.8.7)
pg (0.21.0)

Getting Started
---------------

0. Clone
0. gem install bundler
0. use ruby 2.3.7 (or have `rvm` installed and it will pick the right ruby version)
0. `bundle install`
0. `spring rake db:create db:migrate db:seed`
